# Package: LiveOps Asset Drop

## Buyer

A small mobile or 2D game studio with a launched or funded title, an approved art direction, and a recurring need for new content.

## Default deliverables

- 4–8 new production assets
- 2–4 animation loops or one-shot actions
- 3–6 reusable VFX overlays
- Up to 12 supporting icons or small props
- Layered source files
- Frame sequences or sprite sheets
- Target-engine export package
- Contact sheet and playback capture
- Asset manifest
- QA report

## Required client inputs

- Approved visual references
- Existing production asset for style comparison
- Target canvas and resolution
- Palette and outline rules
- Target engine and version
- Naming and folder requirements
- Gameplay purpose for every asset
- Named approver

## Scope limits

- No new visual identity
- No more than two primary characters
- No complex skeletal rig unless separately scoped
- One consolidated revision round
- Effects limited to agreed canvas and duration

## Acceptance tests

- Style proof approved before batch production
- All files open without corruption
- Transparency and outlines render correctly
- Frame timing matches the brief
- No visible character drift or proportion flicker
- Engine imports preserve pivots, scale, and filtering
- Manifest matches final files
