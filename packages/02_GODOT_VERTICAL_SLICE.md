# Package: Godot Vertical-Slice Art Pack

## Buyer

A funded Godot team approaching a demo, festival, pitch, launch, or publisher milestone.

## Default deliverables

- One environment or encounter slice
- One principal animated subject
- One secondary subject or object set
- Effects and transitions
- SpriteFrames resources
- TileSet or layered scene where applicable
- Pivots, anchors, frame rates, and animation names
- Import settings configured for nearest-neighbor or agreed filtering
- Test scene in a clean Godot project
- Source package, manifest, and QA report

## Supported baseline

- Godot 4.x
- 2D raster and pixel-art workflows
- PNG sequences, sprite sheets, layered scenes, SpriteFrames, TileSets

## Required client inputs

- Exact Godot version
- Reference project or clean integration target
- Canvas size and world scale
- Existing animation naming rules
- Required collision or gameplay reference points
- Style references and target background

## Acceptance tests

- Project opens without missing dependencies
- Test scene runs without import errors
- Animations play at intended speed
- Pivots and scale remain consistent
- Transparent edges are clean
- No unintended filtering or texture bleeding
- Runtime package and editable source package are separate and complete
