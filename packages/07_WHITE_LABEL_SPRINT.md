# Package: White-Label Production Sprint

## Buyer

A motion studio, exhibit agency, learning vendor, game-outsourcing firm, or co-development team with a temporary production gap.

## Default deliverables

- Fixed weekly production capacity
- Client-specific file and naming compliance
- Daily or agreed review drops
- Visual creation, animation, VFX, cleanup, or technical packaging
- Final source and runtime package
- QA report and manifest

## Required client inputs

- Approved statement of work
- Client pipeline guide
- Style references
- Review schedule
- Named producer
- Confidentiality and attribution terms

## Acceptance tests

- Package meets the partner’s client-facing specification
- No studio branding appears unless requested
- All feedback is tracked and resolved
- Files can be forwarded to the end client without repackaging
