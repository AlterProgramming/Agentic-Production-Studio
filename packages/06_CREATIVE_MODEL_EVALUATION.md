# Package: Creative Model Evaluation Pack

## Buyer

A creative studio that is already testing generative models and needs a reproducible production decision.

## Default deliverables

- One tightly defined use case
- Three to five candidate models
- Controlled test prompts or references
- Human evaluation rubric
- Quality, consistency, latency, and cost analysis
- Failure taxonomy
- One adapter or configuration experiment when appropriate
- Deployment recommendation or private endpoint setup
- Model card and provenance record
- Adopt, reject, or continue-testing recommendation

## Required client inputs

- Validated use case
- Test references and usage rights
- Evaluation criteria
- Expected volume and latency
- Privacy and deployment constraints
- Budget ceiling

## Scope limits

- One use case per pilot
- No production-scale custom model training unless separately scoped
- No use of client data without documented permission

## Acceptance tests

- Benchmark can be reproduced
- Each output maps to model, version, prompt, configuration, and source
- Cost and latency are documented
- Recommendation is tied to agreed criteria
- Deployment can be stopped or rolled back safely
