# Master QA Gates

## Visual gate

- Matches approved style proof
- Silhouette remains readable
- Palette and outlines are consistent
- No unintended blur, noise, or texture contamination
- No distorted product, character, or technical details

## Animation gate

- Timing matches brief
- Loop closes correctly when required
- No character or object drift
- No proportion flicker
- No missing frames
- Anticipation, action, impact, and recovery are readable

## VFX gate

- Alpha is clean
- Blend behavior is documented
- Effects do not hide essential gameplay or information
- Effect timing aligns with the base action
- Reusable overlays do not contain unintended background pixels

## Technical gate

- Required dimensions and formats are present
- Naming is consistent
- Pivots and anchors are correct
- Engine import succeeds
- Filtering and compression are correct
- File sizes meet requirements
- Runtime package contains no editable-only dependencies

## Packaging gate

- Source and runtime folders are complete
- Preview matches delivered assets
- Manifest references real files
- Rights and provenance fields are populated
- Revision and approval status are current
- Handoff instructions are usable

## Critical failure conditions

- Missing runtime asset
- Corrupt source file
- Unlicensed or unidentified source material
- Broken engine import
- Material mismatch with approved style
- Incorrect product or technical representation
- Undisclosed AI-assisted transformation where disclosure was required
