# Delivery Standard

## Required top-level folders

- `01_source/`
- `02_runtime/`
- `03_preview/`
- `04_manifest/`
- `05_qa/`
- `06_handoff/`

## Source package

Contains editable working files, references, palettes, layer structure, and any reusable components.

## Runtime package

Contains only files intended for import or publishing. It must not contain scratch files, obsolete versions, or hidden dependencies.

## Preview package

Contains contact sheets, GIFs, MP4 captures, screenshots, or a review build.

## Manifest package

Contains asset IDs, file paths, versions, dimensions, frame data, creation method, rights, and approval status.

## QA package

Contains completed checklists, import results, known limitations, and defect status.

## Handoff package

Contains opening instructions, target software versions, integration notes, extension guidance, and the final scope statement.
