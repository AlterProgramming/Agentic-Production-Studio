# Production Pipeline

## Gate 0 — Scope lock

Required:

- Signed scope or written authorization
- Final asset list
- Complexity tier
- Target formats and platform
- Named approver
- Rights declaration
- Acceptance criteria

## Gate 1 — Source normalization

- Dimensions and scale normalized
- Alpha and background requirements confirmed
- Palette and outline rules captured
- Anchors, pivots, and naming rules documented
- Reference assets stored with source IDs

## Gate 2 — Design proof

- One representative asset or frame
- Motion plan or state diagram
- Effect direction
- Technical export sample
- Client approval before volume production

## Gate 3 — Complete production

- Base artwork complete
- Animation frames complete
- VFX and overlays complete
- Variants created
- Intermediate sources retained

## Gate 4 — Temporal and visual cleanup

- Drift corrected
- Proportions stabilized
- Flicker removed
- Lighting and palette stabilized
- Arcs and impact timing corrected
- Silhouette remains readable

## Gate 5 — Technical integration

- File names normalized
- Runtime exports generated
- Pivots and anchors verified
- Frame rates and loops verified
- Target engine or platform import tested
- Compression and file-size limits checked

## Gate 6 — Package assembly

- Editable source package
- Runtime package
- Manifest
- QA report
- Contact sheet or capture
- Handoff instructions
- Revision and approval record

## Gate 7 — Acceptance

- Buyer reviews the agreed acceptance tests
- Defects are separated from new requests
- One consolidated correction round is completed
- Final approval is recorded
