# Sales Qualification Scorecard

Score each category from 0 to 5.

| Category | Question |
|---|---|
| Active need | Is there a real project or deadline now? |
| Package fit | Does the request match an existing package? |
| Input readiness | Are source files, references, and requirements available? |
| Buyer access | Is the approver known and reachable? |
| Budget | Can the buyer fund at least the minimum pilot? |
| One-week feasibility | Can the outcome be completed in five production days? |
| Repeat potential | Could this become recurring work? |
| Portfolio value | Can the work be shown or credibly anonymized? |
| Technical fit | Is the engine, format, or platform supported? |
| Rights clarity | Are source ownership and AI rules clear? |

## Interpretation

- **40–50:** Priority prospect
- **30–39:** Qualified with one or two risks
- **20–29:** Rescope or sell an audit first
- **Below 20:** Decline
