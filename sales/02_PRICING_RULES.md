# Pricing Rules

## Base pricing model

Use fixed package pricing based on:

- Number of primary assets
- Animation complexity
- Number of effects
- Number of variants and formats
- Integration depth
- Rights and confidentiality requirements
- Revision count
- Delivery speed

## Complexity tiers

### Tier 1 — Controlled
Existing style, simple shapes, short loops, limited layers, standard exports.

### Tier 2 — Production
Moderate detail, multiple states, integrated effects, layered sources, target-platform packaging.

### Tier 3 — Specialist
High-detail cleanup, unusual style constraints, technical accuracy, multiple platforms, private model evaluation, or agency pipeline compliance.

## Commercial rules

- Minimum paid engagement: $1,500
- Standard one-week pilot floor: $3,000
- Rush premium: 25–50%
- Second revision round: priced separately
- New visual direction after approval: new scope
- Source-file exclusion may reduce price only when technically sensible
- Never discount by hiding QA, provenance, or packaging
- Reduce quantity before reducing delivery quality
